# My Portfolio

A modern, responsive personal portfolio website built with React and Vite.

## Description

This portfolio showcases my skills, projects, and experience as a developer. It features a clean design with smooth animations and interactive elements.

## Features

- Responsive design that works on all devices
- Interactive particle background
- Sections for Hero, About, Skills, Projects, and Contact
- Smooth scrolling navigation
- Modern UI components using Tailwind CSS
